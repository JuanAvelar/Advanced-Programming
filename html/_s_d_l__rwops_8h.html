Detailed Description

This file provides a general interface for SDL to read and write data streams.

It can easily be extended to files, memory, etc.

◆ SDL_LoadFile_RW()

DECLSPEC void* SDLCALL SDL_LoadFile_RW ( SDL_RWops *  src,
size_t *  datasize,
int  freesrc 
)

Load all the data from an SDL data stream.

The data is allocated with a zero byte at the end (null terminated)

If datasize is not NULL, it is filled with the size of the data read.

If freesrc is non-zero, the stream will be closed after being read.

The data should be freed with SDL_free().

Returns
the data, or NULL if there was an error.
